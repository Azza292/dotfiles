# Install

## Requires
- https://github.com/twpayne/chezmoi

## Initialize

```shell
# HTTPS
$ chezmoi init https://github.com/azza292/dotfiles

# SSH
$ chezmoi init git@github.com:Azza292/dotfiles.git
```
